This website aims to provide open data standard from Logginwolf. Currently, documentations for general usages of this application is unavailable.

Logginwolf is an application for recording progress of The Werewolves of Miller's Hollow and it's variants. Currently, Logginwolf supports archiving these variants as below:

  • Origin
    • Basic
    • Expansion pack 1: New Moon
    • Expansion pack 2: The Village
  • Chinese (狼人杀/狼人殺)
Each variants has slightly difference regarding on rules, game flow and etc. that the completed information can be found in game mode page.

The program is adapted Flutter as frontend framework, which can be deployed to mobile (Android)1 and desktop (Windows, Linux)1 platforms at once. Although Logginwolf is not an open source program, dependencies derived from this project will keep open and release to pub.dev as well as publishing source codes into GitHub. All repositories will be published under MPL 2.0 license.

Remarks:

  1. macOS and iOS release are not available but reserved for build.

Doubtful information concern

Although origin Werewolf game manual can be found in online, it's Chinese variants is still rely on the corresponded article from Chinese Wikipedia. In additions, it remains uncertainty on making consensus rules due to rapid growth of trends despite the roles are binarized already. Therefore, the adoption of Chinese variant must rely on the Wikipedia pages until alternative sources available with credibility.
